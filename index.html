About Me

Front-end web developer utilizing my background as an aerospace machinist to build clean and bug free experiences for end users. Recently earned a certificate in full stack development for the MERN stack from the University Washington Coding Boot Camp, with new skills in HTML, JavaScript, CSS, React.js, Node.js, and responsive web design. I’m passionate about building applications and learning as much as I can along the way I am a strong problem solver and able to learn new concepts easily. With the projects I develop I aim to make them engaging and easy to use with a focus on mobile development with a clean user experience. I recently worked on a project with four other people to develop a site to get recipes from an API and display them in a fun entertaining way. We also used agile development to improve our project as we built it. I’m excited to build better quality-driven web experiences for users on the web.
